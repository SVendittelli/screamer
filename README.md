## Install

```terminal
npm install
```

### Dependencies

- [node](https://nodejs.org/) - 20.x
- [Vercel CLI](https://vercel.com/docs/cli) (run `vercel login`)

## Usage

```terminal
vercel env pull .env
npm run dev
```

Open <http://localhost:3000> with your browser to see the site.
